# Notes App

Минималистичное приложение для заметок с тегами и поиском.  

# Возможности
- Добавление заметок
- Редактирование заметок
- Удаление заметок
- Теги и фильтрация по ним
- Поиск по тексту заметок
- Сохранение в localStorage

# Технологии
- React + Vite
- useState / useEffect
- CSS

# Запуск проекта
npm install
npm run dev

# Notes App
- Пет проект 

# Демо
- [Live Demo Netlify](https://uniqxxvii-notesapp.netlify.app/)

# Возможности
- Добавление заметок
- Редактирование заметок
- Удаление заметок
- Теги и фильтрация по ним
- Поиск по тексту заметок
- Сохранение в localStorage

# Технологии
- React 
- Vite
- CSS

# Запуск проекта
- npm install
- npm run dev
